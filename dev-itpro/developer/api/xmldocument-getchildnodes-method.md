# GetChildNodes Method
Gets a list containing the child elements for this document, in document order.  
```  
ChildNodes := XmlDocument.GetChildNodes()  
```  
## Return Value
*ChildNodes*  
&emsp;Type: [XmlNodeList](xmlnodelist-class.md)
